# Next.js 入门指南

Next.js 是一个用于生产环境的 React 框架，它提供了许多开箱即用的功能，让开发者能够快速构建高性能的 Web 应用。

## 为什么选择 Next.js？

Next.js 提供了许多优秀的特性：

- **服务器组件**: 默认服务器端渲染，提升性能和 SEO
- **文件系统路由**: 基于文件系统的直观路由
- **数据获取**: 简化的数据获取方法
- **自动代码分割**: 智能的代码分割策略
- **TypeScript 支持**: 一流的 TypeScript 支持

## 快速开始

创建一个新的 Next.js 项目：

\`\`\`bash
npx create-next-app@latest my-app
cd my-app
npm run dev
\`\`\`

## 项目结构

一个典型的 Next.js 项目结构：

\`\`\`
my-app/
├── app/
│ ├── layout.tsx
│ └── page.tsx
├── components/
├── public/
└── package.json
\`\`\`

## 路由系统

Next.js 13+ 使用基于文件系统的路由：

\`\`\`typescript
// app/page.tsx
export default function Home() {
return <h1>Welcome to Next.js!</h1>
}
\`\`\`

## 数据获取

使用服务器组件获取数据：

\`\`\`typescript
async function getData() {
const res = await fetch('https://api.example.com/data')
return res.json()
}

export default async function Page() {
  const data = await getData()
  return <main>{/* 使用数据 */}</main>
}
\`\`\`
